
# Yoin

This is a very simple terminal tool to create and manage sound layers to create a zen like sound environment to help you focus on your work.


## Features

- Cross platform

- Adding SoundLayers
- Muting SoundLayers
- Setting Volume of separate SoundLayers

- ToneLayer (Frequency, Support for LFO)
- WhiteNoise (Support for LFO)
- BrownNoise (Support for LFO)



## Installation


### Linux

Install my-project with curl

```bash
CREATE CURL INSTALL SCRIPT
```
You can remove easily by removing the whole directory of this project

```bash
rm -r ~/yoin
```
